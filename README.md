# 💀 when_you-death

A fun and interactive web application that estimates how long a person might live based on the information they provide.

> ⚠️ This project is made purely for entertainment purposes and should not be taken seriously.

---

## 🚀 Features

- 📝 User-friendly form to collect basic personal information  
- 🤖 Calculates an estimated lifespan based on input data  
- 🎯 Instant results with a fun and engaging output  
- 🌐 Simple and clean web interface  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## 📸 How It Works

1. User opens the web app  
2. Fills in personal details (like age, habits, etc.)  
3. Clicks on the submit button  
4. The system processes the data  
5. Displays an estimated lifespan result  

---

## ⚠️ Disclaimer

This project is created for **fun and educational purposes only**.  
The results are not scientifically accurate and should not be used for real-life decisions.
